# Auto-repo (RepoFlow)

RepoFlow adalah web app gaya native yang dioptimalkan untuk **HP** dan tetap nyaman di **PC**. Alurnya: sambutan -> masuk aplikasi -> upload file -> review hasil ekstrak -> push ke GitHub.

## Fitur utama

- **Dashboard**: statistik repo, grafik aktivitas push 7 hari, dan daftar repository terbaru.
- **Upload All File**: dukung semua jenis file (Java, Python, TypeScript, HTML, dll). Bisa upload file tunggal atau ZIP.
- **Preview Ekstrak ZIP**: lihat daftar file hasil ekstrak, lalu hapus file yang tidak perlu sebelum push.
- **Push ke GitHub**: bukan deploy; membuat repository baru lalu push file satu per satu.
- **Tools**: cari repository, salin URL, buka repo, dan hapus repo.
- **Info Web**: profil web, profil developer, dan seluruh link komunitas/sosial media.
- **Onboarding GitHub**: user ditanya dulu apakah sudah punya akun GitHub sebelum diminta token.
- **SEO/Share Meta**: favicon + banner Open Graph/Twitter agar preview link tampil rapi di WhatsApp/X.
